# RI4_Modularizacao_ES6
Projeto de POO em JavaScript (RI 4) que introduz a modularização ES6 (import/export). Reutilizando o contexto do RI 3, este exercício separa as classes (Empresa, Cliente, Telefone, Endereco) em um módulo principal e as importa em um módulo de teste (consumidor). Garante o uso de classes, atributos privados e métodos de acesso e formatação.
